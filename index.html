What is Web 3.0 and how is it different from Web 1.0 and Web 2.0?

In the case of Web 1.0 it was only one way communication via static websites, whereas Web 2.0 allows the user to interact with the website instead of only viewing or consuming the content all while live.

What are the functional differences between the front-end of a web application and its back-end?

The frontend is compiled by the browser through html and css code, encoded instructions to the browsers describing content to be displayed and css code are instructions how the content should be styled. The Backend is processed on the server, like tools to access data in a database or communicate or process information.
